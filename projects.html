PROJECT 01

AI Tutoring Platform

Interactive Python Learning with LLM-Powered Assistance

Built a comprehensive AI-powered tutoring platform designed to teach Python programming through an interactive, hands-on approach. The platform combines structured lessons, practice problems, and real-time AI tutoring to create an engaging learning experience. Students can visualize programming concepts through pseudocode and flowcharts, practice with guided exercises, and receive personalized help from an integrated LLM assistant.

Key Features

Guided Lessons
Structured Python curriculum with progressive difficulty levels
Interactive Practice
Real-time code execution with instant feedback using Pyodide
Visual Learning
Pseudocode visualization and flowcharts using Graphviz
LLM Tutoring
Personalized AI assistance powered by Ollama
Progress Tracking
MongoDB-based user progress and performance analytics
Clean Data Pipeline
NLP-based content preprocessing for quality assurance

Technologies Used

JavaScript Python MongoDB Ollama Pyodide Docker HTML/CSS RESTful APIs Graphviz
Impact & Learning

This project was developed during my internship at ICFOSS as part of their e-Governance initiatives. It enhanced my skills in full-stack development, NLP data processing, and integrating AI/ML models into production applications. The platform demonstrates my ability to build scalable educational tools with modern web technologies.

PROJECT 02

Automatic Vehicle Speed Control System

IoT-Based Smart Traffic Management Solution

Designed and implemented an ESP32-based IoT system for intelligent vehicle speed management in restricted zones such as school areas, hospitals, and residential neighborhoods. The system uses RFID technology to detect vehicles entering controlled zones and automatically limits their speed through wireless communication. A comprehensive monitoring solution includes both Android and web applications for real-time tracking and alert management.

Key Features

Zone Detection
RFID-based automatic detection of restricted zones
Speed Control
Automatic speed limitation via RF module communication
Real-Time Monitoring
Web dashboard for live vehicle tracking and zone status
Mobile App
Android application for authorities to manage and monitor
Alert System
Bluetooth-based alerts and violation notifications
Scalable Design
Modular architecture for easy zone expansion

Technologies Used

ESP32 RFID RF Modules Bluetooth Android HTML/CSS JavaScript IoT Protocols
Impact & Learning

This project showcases my expertise in IoT development, embedded systems programming, and full-stack application development. It addresses a real-world problem of traffic safety in restricted zones, demonstrating my ability to design end-to-end solutions combining hardware and software components. The project strengthened my skills in wireless communication protocols and cross-platform application development.

PROJECT 03

Blood Donation System

Web-Based Platform Connecting Donors with Recipients

Developed a comprehensive web-based blood donation management system to bridge the gap between blood donors and recipients. The platform streamlines the entire donation process, from donor registration and blood type matching to donation scheduling and tracking. Built with a focus on user experience and data management, the system helps healthcare facilities and blood banks efficiently manage their donor database and coordinate emergency blood requirements.

Key Features

Donor Registration
Easy signup process with comprehensive donor profiles
Smart Search
Search donors by blood type, location, and availability
Donation Tracking
Complete history of donations with dates and centers
Responsive Design
Mobile-friendly interface using Bootstrap framework
Admin Dashboard
Comprehensive management panel for blood banks
Request Management
Urgent blood requirement posting and matching system

Technologies Used

HTML CSS Bootstrap JavaScript PHP MySQL
Impact & Learning

This project demonstrates my proficiency in full-stack web development and database design. It addresses a critical healthcare need by making blood donation more accessible and organized. Through this project, I gained extensive experience in PHP backend development, MySQL database management, and creating responsive user interfaces. The system's focus on connecting people for a life-saving cause makes it particularly meaningful.
